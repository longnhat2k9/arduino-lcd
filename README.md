# Arduino LCD Display 4x20

## Description
Learn Project. This project made for learn how to use LCD Display with arduino board.
## Component List
```
1. Arduino Nano
2. LCD Display 4x20
3. Ultrasonic Sensor (4 pin)
4. Gas Sensor
```
